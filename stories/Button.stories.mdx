# Button

A button is a field or area of text that communicates and triggers user actions when clicked or selected.
`Buttons` can be used to instantly communicate and trigger actions that a user can perform in an application,
such as For example, submitting a form, canceling an operation, or creating a new object.

## Primary variant button

Primary buttons are the maximum visually outstanding variant. Use for the maximum critical name to action.
